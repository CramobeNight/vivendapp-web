# VivenDapp Web — vivendapp.io

Landing page oficial del protocolo de tokenización inmobiliaria VivenDapp.

## Stack
- HTML5 / CSS3 / Vanilla JS
- Sin dependencias externas (excepto Google Fonts)
- Deploy automático vía Netlify

## Estructura
```
/
├── index.html       # Landing page completa (single file)
├── netlify.toml     # Configuración de deploy
├── whitepaper.pdf   # Whitepaper del protocolo (agregar)
└── og-image.jpg     # Imagen para redes sociales (agregar)
```

## Deploy
Conectado a Netlify → `vivendapp.io`

Cualquier push a `main` despliega automáticamente en producción.
